## ***Projeto de uma aplicação web para um Petshop - Fase 1***

Fase 1 do Projeto da disciplina Fundamentos de Sistemas Web: desenvolver um sistema web para um petshop, focando apenas na construção e arquitetura das páginas HTML, sem customização de CSS, Boostrap e/ou funções de JS.

O projeto é estruturado com o viés de mostrar conhecimento e compreensão da estrutura básica de sistemas web, contendo: 
  ### *Header:*
  Informações do petshop, links/menu de navegação para as páginas.
  
  ### *Body:*
  Visualização de dois produtos de cada uma das 3 categorias (foto, descrição, valor) e os serviços oferecidos pelo petshop (descrição, valor).
  
  ### *Footer:*
  Informações legais de autoria, contato, links de navegação, etc.
